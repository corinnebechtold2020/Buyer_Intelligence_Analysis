LSC Buyer Journey Intelligence Engine

This tool processes Life Science Connect exports to build buyer-journey intelligence and produce a multi-tab Excel report.

What the tool does

  • Accepts LSC XLSX or CSV exports
  • Validates required fields and processes large volumes (~40k rows)
  • Assigns buyer-journey stages and aggregates one row per individual
  • Generates Company_Combined, Company_Summary, Individuals, Product_Map, Sales_Hot_List
  • Produces a color-coded Excel report ready for download

How to run locally

Install Python 3.10 and dependencies, then run Streamlit:

pip install -r requirements.txt
streamlit run app.py --server.address 0.0.0.0 --server.port 8000

How to launch in GitHub Codespaces

This repository includes a `.devcontainer` configuration. Open in Codespaces or a Dev Container and the app will be started automatically at port 8000.
